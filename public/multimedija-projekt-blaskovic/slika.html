Detaljan postupak izrade

  • Promjena originalne slike iz grayscale moda u RGB mod, Image → mode → RGB
  • Dodavanje novog transparentnog layera, preimenovanje u cartoon layer
  • Rad na originalnom layeru, odabir Path alata
  • Obilaženje cijelog portreta kako bi se dobio odabir
  • Kreiranje odabira, desni klik → Select → From Path
  • Odabir cartoon layera
  • Odabir bucket alata, odabir boje kože, nanešavanje boje na odabir
  • Skrivanje cartoon layera, te nastavak..
  • Ponavljanje postupka ovaj put za kosu
  • Kreiranja odabira, bucket tool nanošenje tamno sive gotov crne boje na cartoon layer
  • Ponavljanje postupka za obrve, oči, nos, brkove, usta, uho te naposlijetku i vilicu
  • Za vilicu i nos nisam radio puno kruženje oko njih budući da nije dobro izgledao, zato sam radio samo manje dijelove, (kako bi se dobio dojam dubine)
  • Kada sam završio, dodao sam još jedan transparentni layer pomoću kojeg ću raditi pozadinu
  • Sakrio sam originalni layer, dodao sam tirkiznu pozadinu u novi layer bucket alatom
  • Exportao sam jednu ovakvu verziju slike

    • Bonus:

  • Odlučio sam se još malo igrati layerima te sam postavio originalni layer iznad cartoon layera, promijenio sam mu mod iz normala u overlay

    • Objašnjenje kako funkcionira overlay mod u GIMPU: Overlay mode inverts the pixel value of the lower layer, multiplies it by two times the pixel value of the upper layer, adds that to the original pixel value of the lower layer, divides by 255, and then multiplies by the pixel value of the original lower layer and divides by 255 again. It darkens the image, but not as much as with “Multiply” mode.

  • Na originalni sam layer dodao Alpha kanal. Alpha kanal reprezentira transparentnost slike.
  • Naposlijetku sam smanjio opacity originalnog layera na oko 25% kako bi dobio efekt sijene, teksture i dubine na cartoon verziji
